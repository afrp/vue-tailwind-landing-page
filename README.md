# vue-tailwind-landing-page

using 
- npm version 8.11.0 
- node version 16.15.1
- vue version 3.3.4
- tailwindcss version 3.4.1
- vite version 4.4.5

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
